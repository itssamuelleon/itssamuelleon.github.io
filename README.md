## 🚀 Sobre el proyecto
Este repositorio contiene el código fuente de mi *Landing Page* profesional. Fue diseñada para ser rápida, responsiva y directa, mostrando mis proyectos más destacados, experiencia laboral y stack tecnológico. 

Está alojada de manera gratuita y continua utilizando **GitHub Pages**.

## 🛠️ Tecnologías Utilizadas
El portafolio fue construido utilizando tecnologías ligeras y modernas, sin necesidad de frameworks pesados:
* **HTML5:** Estructura semántica.
* **Tailwind CSS:** Estilos y diseño responsivo implementados vía CDN.
* **FontAwesome:** Iconografía vectorial.

## 💻 Desarrollo Local
Si deseas descargar o modificar este código en tu entorno local, no necesitas instalar dependencias de Node.js ni configurar un servidor.
1. Clona este repositorio: `git clone https://github.com/itssamuelleon/itssamuelleon.github.io.git`
2. Abre el archivo `index.html` directamente en cualquier navegador web.
